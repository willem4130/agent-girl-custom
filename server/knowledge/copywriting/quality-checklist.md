# Copy Quality Checklist

Use this checklist to evaluate all copy before delivery.

## Gate 1: Authenticity (Minimum 8/10)

### Forbidden Words (Auto-Fail if Present)
- [ ] leverage
- [ ] unlock
- [ ] dive deep
- [ ] game-changer
- [ ] cutting-edge
- [ ] synergy
- [ ] ecosystem
- [ ] paradigm shift
- [ ] next-level
- [ ] best-in-class
- [ ] holistic
- [ ] streamline
- [ ] actionable insights
- [ ] at the end of the day
- [ ] move the needle
- [ ] low-hanging fruit
- [ ] think outside the box
- [ ] circle back
- [ ] touch base

### Required Elements
- [ ] Contractions used naturally (you're, we've, that's, it's)
- [ ] Active voice throughout (not "was developed" but "we developed")
- [ ] Strong verbs (not "utilize" but "use")
- [ ] Conversational tone (sounds like a person, not a press release)
- [ ] No corporate jargon or buzzwords

### Red Flags
- [ ] No excessive dashes pretending to be thoughtful pauses
- [ ] No overly formal sentence structures
- [ ] No AI-typical hedging ("It's important to note that...")
- [ ] No generic filler phrases ("In today's world...")

## Gate 2: Specificity (Minimum 8/10)

### Numbers and Data
- [ ] Specific numbers where claims are made (not "many" but "7,432")
- [ ] Precise timeframes (not "soon" but "within 48 hours")
- [ ] Concrete metrics (not "improved" but "increased by 23%")
- [ ] Real examples, not hypotheticals

### Details
- [ ] Named examples when possible
- [ ] Tangible outcomes described
- [ ] Sensory or vivid language
- [ ] Proof points for claims

### Avoid
- [ ] Vague superlatives ("the best", "amazing", "incredible")
- [ ] Unsubstantiated claims
- [ ] Generic descriptions that fit any company
- [ ] Empty promises without specifics

## Gate 3: Tone Alignment (Minimum 8/10)

### Brand Voice Dimensions
Check against brand voice profile:

- [ ] Formality level matches (1-10 scale)
- [ ] Humor/seriousness balance matches
- [ ] Energy level matches (calm vs enthusiastic)
- [ ] Authority level matches (peer vs expert)

### Consistency
- [ ] Vocabulary matches brand patterns
- [ ] Sentence structure aligns with brand style
- [ ] Emoji usage appropriate for brand
- [ ] CTA style matches brand approach

### Language
- [ ] Correct language (Dutch vs English)
- [ ] Native expressions (not translations)
- [ ] Local references if appropriate

## Gate 4: Trigger Deployment (Minimum 7/10)

### Framework Application
- [ ] Clear copywriting framework applied (PAS, AIDA, BAB, etc.)
- [ ] Framework executed properly (all elements present)
- [ ] Framework appropriate for content type and audience

### Psychological Triggers
- [ ] At least one trigger clearly deployed
- [ ] Trigger appropriate for the context
- [ ] Trigger intensity matches audience relationship stage
- [ ] Not manipulative or excessive

### Integration
- [ ] Triggers feel natural, not forced
- [ ] Multiple elements work together coherently
- [ ] CTA aligns with trigger deployment

## Overall Quality Check

### Structure
- [ ] Strong opening hook (first line grabs attention)
- [ ] Logical flow from start to finish
- [ ] Single clear message/goal
- [ ] Compelling call-to-action

### Readability
- [ ] Appropriate length for medium/platform
- [ ] Scannable format (short paragraphs, clear sections)
- [ ] No walls of text
- [ ] Visual hierarchy if applicable

### Polish
- [ ] Grammar and spelling correct
- [ ] No awkward phrasing
- [ ] Consistent formatting
- [ ] Platform-appropriate (character limits, hashtags, etc.)

## Scoring Guide

### For Each Gate:
- **10/10**: Exceptional, exceeds all requirements
- **9/10**: Excellent, minor improvements possible
- **8/10**: Good, meets professional standards
- **7/10**: Acceptable, some issues to address
- **6/10**: Below standard, revisions needed
- **5/10 or below**: Significant issues, major revision required

### Final Verdict:
- **APPROVED**: All gates ≥8/10 (or ≥7/10 for triggers)
- **NEEDS REVISION**: Any gate below threshold
- **REJECTED**: Multiple gates significantly below standard

## Quick Self-Check Questions

Before submitting copy, ask:

1. Would I share this if it was about me?
2. Does this sound like a real person wrote it?
3. Is there specific proof for every claim?
4. Would the target audience recognize themselves?
5. Is the CTA clear and compelling?
6. Would I click/engage with this?

If any answer is "no" or "maybe", revise before submitting.
