# /analyze - Analyze Copy or Competitor Content

## What This Does
Performs deep analysis of existing copy to extract insights, identify patterns, and suggest improvements.

## How to Use
/analyze [paste copy or provide URL]

Example:
/analyze Check out this landing page: https://example.com/product

## Process
1. Read/scrape the provided content
2. Analyze tone dimensions (formality, humor, energy, authority)
3. Identify copywriting frameworks used
4. Detect psychological triggers deployed
5. Evaluate against quality standards
6. Provide actionable improvement suggestions
