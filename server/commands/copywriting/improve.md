# /improve - Improve Existing Copy

## What This Does
Takes existing copy and enhances it using proven techniques while preserving the original intent.

## How to Use
/improve [paste the copy to improve]

Example:
/improve "We offer great products at competitive prices. Contact us today!"

## Process
1. Analyze the original copy
2. Identify specific weaknesses
3. Apply targeted improvements
4. Generate 3 improved versions (light, medium, bold)
5. Explain each change made

## Improvement Levels

### Light Touch
- Fix obvious issues
- Add specificity where vague
- Improve word choice
- Maintains original structure

### Medium Rewrite
- Apply copywriting framework
- Add psychological trigger
- Improve flow and rhythm
- May restructure slightly

### Bold Transformation
- Complete reframe if needed
- Different angle/approach
- Maximize persuasion
- May significantly change structure

## Output Format

### Original Analysis
- **Main weakness**: [What's holding it back]
- **Missing elements**: [What's absent]
- **Tone assessment**: [Current vs optimal]

### Light Touch Version
[Improved copy]
*Changes: [List of specific changes]*

### Medium Rewrite Version
[Improved copy]
*Changes: [List of specific changes]*
*Framework applied: [Name]*

### Bold Transformation Version
[Improved copy]
*Changes: [List of specific changes]*
*New angle: [Explanation]*

### Recommendation
[Which version to use and why]

## Common Improvements Applied
- Replace passive voice with active
- Add specificity to vague claims
- Include social proof
- Strengthen weak verbs
- Remove filler words
- Add curiosity hooks
- Clarify value proposition
- Improve call-to-action
