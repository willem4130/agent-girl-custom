# /social - Generate Social Media Post

## What This Does
Creates 3-5 variations of a social media post using different copywriting frameworks.

## How to Use
/social [topic or brief]

Example:
/social New eco-friendly packaging launch announcement

## Process
1. Review brand voice profile from session context (if available)
2. Select appropriate framework (PAS, AIDA, Hook-based, or Star-Story-Solution)
3. Generate variations using different psychological triggers
4. Self-evaluate each variation against quality checklist
5. Present options with framework and trigger explanations

## Output Format
For each variation:
- **Framework**: [Name of framework used]
- **Trigger**: [Primary psychological trigger deployed]
- **Copy**: [The actual social media copy]
- **Quality Score**: [Self-assessment out of 10]

## Quality Checklist Applied
- No AI jargon (leverage, unlock, dive deep, game-changer)
- Contractions used naturally (you're, we've, that's)
- Specific numbers/examples included where appropriate
- Matches brand's language (Dutch or English natively)
- Call-to-action appropriate for platform
- Psychological trigger clearly applied

## Platform Considerations
- Instagram: Visual-first, emoji-friendly, 2200 char limit
- LinkedIn: Professional tone, thought leadership angle
- Facebook: Conversational, community-focused
- X/Twitter: Punchy, 280 char limit, thread potential
