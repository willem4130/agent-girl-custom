# Copywriting Project

This workspace is configured for brand-aligned copywriting.

## Brand Configuration

Configure your brand references in the chat to enable tone matching:
- Website URL (primary brand voice reference)
- Instagram profile (social media patterns)
- Facebook page (community engagement style)
- LinkedIn company page (professional tone)

## Available Commands

```bash
/social [topic]      # Generate social media post variations
/newsletter [topic]  # Create newsletter content with subject lines
/headlines [topic]   # Generate 10+ headline variations
/analyze [copy/URL]  # Analyze existing copy or competitors
/improve [copy]      # Enhance existing copy (3 levels)
```

## Copywriting Frameworks

### PAS (Problem-Agitate-Solution)
Best for: Pain point content, problem-aware audiences
1. Identify the problem
2. Amplify the frustration
3. Present your solution

### AIDA (Attention-Interest-Desire-Action)
Best for: Cold audiences, general marketing
1. Grab attention with a hook
2. Build interest with benefits
3. Create desire with proof
4. Call to action

### BAB (Before-After-Bridge)
Best for: Transformation content, testimonials
1. Show current situation
2. Paint the ideal future
3. Bridge with your solution

### Star-Story-Solution
Best for: Narrative content, brand storytelling
1. Introduce relatable character
2. Tell their journey
3. Reveal the solution

## Psychological Triggers

**Low intensity** (use freely):
- Curiosity gaps
- Specificity with numbers
- Social proof

**Medium intensity** (use strategically):
- Authority citations
- Reciprocity
- Scarcity

**High intensity** (use sparingly):
- FOMO (only genuine)
- Identity appeal
- Real urgency

## Quality Standards

Every piece of copy is self-evaluated against:
- Authenticity (no AI jargon): 8+/10
- Specificity (concrete details): 8+/10
- Tone alignment (brand match): 8+/10
- Trigger deployment: 7+/10

## Language Support

Native support for:
- Dutch (Nederlandse copy)
- English

Use native expressions, not translations. Match the user's language automatically.

## Output Format

All copy requests generate 3-5 variations with:
- Framework used
- Primary trigger deployed
- Brief rationale
- Quality self-assessment
